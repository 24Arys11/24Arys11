\section{Projects}

\project{\emph{Arduino} Matrix Game}{https://github.com/24Arys11/Robotica/tree/master/The\%20Matrix\%20Project}	% 1. name, 2. link, 3. description
{
\emph{Gaming console} with two controllers equipped with a \emph{3D printed} case made of several components modeled in \emph{Autodesk Fusion 360 (CAD)}. Runs a retro version of the \emph{atomic bomberman} game in single player mode (vs \emph{AI}) or in two players (PvP).
}

\project{\emph{Machine Learning} Classification}{https://github.com/24Arys11/Artificial-Intelligence-Machine-Learning-and-Deep-Learning/tree/main/Project\%20ML\%20-\%20Computer\%20Tomography\%20Scan\%20Classification}	
{
Kaggle competition for \emph{image classification} of pulmonary tomography images with three types of blood vessels: native, arterial and venous. I used Support Vector Machine (\emph{SVM}), Convolutional Neural Networks (\emph{CNN}) and I modified the \emph{Resnext101\_32x8} model by adding four fully connected layers. I obtained the accuracy of {74.29\%}, well above the \emph{required 39.38\%}.
}

\project{\emph{Arduino} Bike}{https://github.com/24Arys11/Robotica/tree/master/Motorbike\%20project}	% 1. name, 2. link, 3. description
{
A 3D printed \emph{motorcycle} (with \remph{headlights}, \remph{taillights}, \remph{signals}, etc.) that \emph{maintains its balance} (using PID), controlled by \emph{radio}. The \emph{remote} had a Joystick (direction) and a button (horn).
}

\project{\emph{AI} Pygame}{https://github.com/24Arys11/Artificial-Intelligence-Machine-Learning-and-Deep-Learning/tree/main/Project\%20AI\%20-\%20Game\%20(Min-Max\%20\%26\%20Alpha-beta)}	% 1. name, 2. link, 3. description
